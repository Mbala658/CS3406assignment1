## Project Description

This project implements the Modified Gale-Shapley Stable Matching Algorithm for assigning medical residents to hospitals with multiple available positions.

The program takes as input a list of hospitals, their capacities, and preference lists, along with the residents' preference lists. It produces a stable assignment of residents to hospitals while allowing for unmatched residents when there are more residents than available hospital positions.
